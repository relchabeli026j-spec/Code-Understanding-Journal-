# Code-Understanding-Journal-
Overview

The application is a command-line Task Manager that allows users to create, view, and manage tasks. It interacts with the user through terminal input and output.
Program Entry Point

The program starts in main.py, where the main loop is defined. This loop continuously displays options to the user and processes their input.
Core Features

The application includes the following functionality:

Adding a task
Viewing all tasks
Deleting or updating tasks
Code Structure

The code is divided into functions that handle specific actions. This improves readability and makes the system easier to maintain.
Data Management

Tasks are stored in:

a list (temporary), or
a file (for persistence)

This allows the application to keep track of user tasks.
Program Flow
Program starts
Menu is displayed
User selects an option
Corresponding function runs
Loop continues until exit
Observations
The program uses loops to keep running
Functions separate logic clearly
Input/output is handled via terminal
Improvements
